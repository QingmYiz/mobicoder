## MobiCoder 改造完成

将 openclaw-termux 全面改造为 MobiCoder - 手机 AI 编程 Agent。

### 核心变更
- **去掉 OpenClaw 依赖**: 删除所有 openclaw gateway 相关逻辑，包括 openclaw.json 配置、gateway 启动/停止、web dashboard、onboarding、configure 页面
- **新建 Agent Server**: agent-server/ 目录，基于 Express + WebSocket，提供 chat/agent/files/terminal/build/capabilities API，端口 18790
- **重命名**: OpenClaw → MobiCoder, Gateway → Agent, openclaw.json → mobicoder.json
- **新增 5 个 Flutter 页面**: chat_screen, agent_screen, file_manager_screen, project_screen, build_screen
- **重做主面板**: dashboard_screen.dart 新增 AI Chat、Agent Mode、File Manager、Projects、Build APK 入口
- **保留全部手机能力**: 8 种能力 20 条命令（相机/闪光灯/GPS/屏幕录制/传感器/振动/串口/Canvas）
- **保留 SSH**: 远程管理功能不变
- **APK 模板**: agent-server/templates/android-app/ 提供基础 Android 项目模板

### 文件变更统计
- 新建约 20 个文件（agent-server/ 全部 + 5 个新 screen + 3 个 model/provider/service）
- 修改约 15 个文件（constants, app, main, dashboard, logs, settings, splash, gateway_controls, node_provider, bootstrap_service, setup_state, CLI, package.json, pubspec.yaml）
- 删除 3 个页面文件（web_dashboard, onboarding, configure 的内容被替换）

### 关键架构
- Flutter App → NativeBridge (MethodChannel) → Kotlin → proot Ubuntu → MobiCoder Agent Server (Node.js)
- Agent Server 直连 OpenAI/Anthropic SDK，不经过 OpenClaw 网关
- Chat 模式: http SSE streaming
- Agent 模式: WebSocket，支持 think→act→observe 循环
- 文件操作、终端、APK 构建均通过 Agent Server API
